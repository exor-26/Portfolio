# Aditya Portfolio

A cinematic portfolio website for Aditya, a software architect building production systems across Android, AI, ERP, and the web.

## Overview

This project is a single-page portfolio built with React, Vite, and Tailwind CSS. It combines a dark premium visual language with interactive UI details such as a custom cursor, spotlight lighting, floating gradient orbs, glassmorphism cards, animated counters, typewriter hero text, scroll-based section tracking, and project card motion.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Custom CSS animations and interactions

## Features

- Responsive layout for mobile, tablet, desktop, and large screens
- Glassmorphism interface with dark cinematic styling
- Animated hero section with typewriter effect
- Scroll progress bar and active navigation highlighting
- Interactive project cards with hover depth and glow treatment
- Skills ticker and animated counters
- Contact section with direct links to email, GitHub, and Instagram

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Customization

Most of the content is defined directly inside `src/AdityaPortfolio.jsx`, including:

- hero copy
- project data
- skills
- contact links
- visual styles

This keeps the portfolio self-contained and easy to adapt.

## Deployment

The project can be deployed for free using any static hosting provider that supports Vite builds, including:

- Netlify
- Vercel
- Cloudflare Pages
- GitHub Pages

Build command:

```bash
npm run build
```

Output directory:

```text
dist
```
